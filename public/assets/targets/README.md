Thư mục này chứa file .mind (compiled image targets).

Cách tạo file targets.mind:
1. Truy cập https://hiukim.github.io/mind-ar-js-doc/tools/compile
2. Upload ảnh mục tiêu (image target) theo thứ tự
3. Nhấn "Start" để biên dịch
4. Tải file .mind về và đặt vào thư mục này với tên "targets.mind"

Lưu ý:
- Ảnh nên có nhiều chi tiết, tương phản cao
- Tránh ảnh đối xứng hoặc ít đặc trưng
- Kích thước khuyến nghị: 400×400 px trở lên
