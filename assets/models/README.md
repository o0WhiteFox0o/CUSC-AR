Thư mục này chứa các model 3D (GLTF / GLB).

Định dạng hỗ trợ:
- .glb (khuyên dùng – binary, nhỏ gọn)
- .gltf + .bin + textures

Nguồn model miễn phí:
- https://sketchfab.com (chọn license CC)
- https://poly.pizza
- https://kenney.nl/assets

Lưu ý:
- Model nên dưới 5MB để tải nhanh trên mobile
- Nên dùng Draco compression để giảm kích thước
